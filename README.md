# CodeVector Internship Task

This is a full-stack product browser app made using **React + Vite** for frontend and **Node.js + Express + MongoDB** for backend.

## Live Links

- LIVE URL: https://product-browser-frontend-xi.vercel.app/

## Features

- View products in a clean UI
- Filter products by category
- Pagination for product listing
- Add new products
- Update existing products
- Shows product creation time

## Tech Stack

- Frontend: React, Vite, CSS
- Backend: Node.js, Express
- Database: MongoDB

## Project Structure

- `frontend/` - React app
- `backend/` - Express API

## How to Run Locally

1. Clone the repo  
2. Install dependencies in both frontend and backend folders:
   - `npm install`
3. Start backend:
   - `npm run dev`
4. Start frontend:
   - `npm run dev`

This project was built as an internship task to practice full-stack development.
